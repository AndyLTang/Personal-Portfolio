About // A web app written in Ruby on Rails that allows users to create accounts to generate, view, and vote on polls to share with other players in a game called Shadowverse.

Link // ShadowPoll.net

Date // Feb 2017 - Current

Technologies // Ruby on Rails Devise MVC HTML CSS Bootstrap Javascript JQuery

Highlights //

  • Developed a web app using the Ruby on Rails MVC web framework to dynamically generate the content of each page.
  • Created custom forms for polls which allow users to add/remove any number of nested resources of a poll as they want (limited to 15) by dynamically adding the nested forms.
  • Incorporated the Devise gem to allow users to create accounts to prevent bots from spamming the polls.
