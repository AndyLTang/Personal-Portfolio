Shadow Poll

ABOUT

A web app written in Ruby on Rails that allows users to create accounts to generate, view, and vote on polls to share with other players for a game called Shadowverse.

LINKS

Website: ShadowPoll.net

Repo: GitHub

DATE

Feb 2017 - Current

TECHNOLOGIES

Ruby on Rails RubyGems MVC HTML CSS Bootstrap Javascript jQuery JSON AWS EC2 Route 53 Git Phusion Passenger

HIGHLIGHTS

  • Developed a web app using the Ruby on Rails MVC web framework to dynamically generate the content of each page.
  • Deployed to AWS EC2 in a Production Environment by using Phusion Passenger to redirect HTTP requests to the app server.
  • Created custom forms for polls which allow users to add/remove any number of nested resources of a poll as they want (limited to 15) by dynamically adding the nested forms.
  • Incorporated the Devise RubyGem to allow users to create accounts to prevent bots from spamming the polls.
  • Seeded data to the database by parsing a JSON file as well as removing any invalid entries before adding it as a record.
