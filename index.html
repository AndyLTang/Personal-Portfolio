PROJECTS

What I have done

Click the images to view some of my past works.

Hybrid Image

About // An offline Java application which allowed users to upload a pair of pictures in order to create the hybrid image effect.

  • Developed the algorithm to create Hybrid Images
  • Implemented the user interface using Java's Swing Library which allows users to upload their own pair of photos to create a hybridized image.

Watchermelon

About // An iPhone app which allows local neighborhood blockwatchers to send reports to a database where the police processes the information.

  • Prototyped the user interface on Fluid UI
  • Drew the interface elements as scalable vectors using Illustrator
  • Mocked a demo screenshot of the app icon created in Photoshop using smart objects

Amazon Web Services

About // Team project in which we used Amazon's web services to implement the REST API to simulate a server processing client requests and handling erroneous cases.

  • Connected to the servers using the Python boto interface to gain access to features including Elastic Compute Cloud (EC2), DynamoDB, and Simple Queue Service (SQS).
  • Implemented a scheduling algorithm with zookeeper to keep the message order consistent across EC2 instances.

SideTube

About // A personal project to improve YouTube's user interface. It is used in conjunction with any Javascript injecting browser (such as Tampermonkey for Chrome) extension to modify the appearance of the site.

  • Manipulated CSS of Youtube's webpage using JQuery to rearrange elements to visually flowing positions.
  • Hosted script on GitHub to remotely distribute the script.

ABOUT ME

Who I am

Hi, I am Andy Tang.

I am a software developer. I graduated from Simon Fraser University with a Bachelor of Science in Computing Science. However, having deep interest in digital art from a young age, I have a history of small personal creations in digital art. This other half of me has compelled me to mix together the two things I love: coding and digital art. Many see computers as logical pieces of electronic, but to me they are a canvas where we paint with 1's and 0's instead of a paintbrush.
